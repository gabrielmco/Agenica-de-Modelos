# HYPERVISION — Agência Criativa de Design 💎

![HYPERVISION Banner](https://via.placeholder.com/1200x400/000000/FFFFFF?text=HYPERVISION)

Repositório oficial da **HYPERVISION**, uma agência de luxo focada em transcrever a essência de marcas em sistemas visuais de alto impacto. 

## ✨ Funcionalidades (Awwwards-Level Polish)

- **UX de Luxo**: Micro-interações em cada hover, botões com feedback tátil e transições fluidas.
- **GSAP Motion Architecture**: Revelação de texto palavra por palavra, scroll parallax orgânico e loops de animação de alto desempenho.
- **Performance Extrema**:
  - Imagens convertidas de PNG para **WebP** (~95% de compressão).
  - **Lazy Loading** via IntersectionObserver para imagens de fundo.
  - Carregamento de fontes não-bloqueante.
  - Zero dependências pesadas (Vanilla JS + GSAP).
- **SEO & Social**: Pronto para indexação com JSON-LD, Open Graph e Twitter Cards.
- **Acessibilidade**: 100% navegável via teclado com suporte ARIA completo.

## 🛠 Tech Stack

- **Linguagem**: HTML5 Semântico / Vanilla JavaScript (ES6+)
- **Estilos**: SCSS (Arquitetura Modular)
- **Animações**: GSAP 3 (ScrollTrigger)
- **Host**: Vercel (Pronto para Deploy)

## 🚀 Como Executar Localmente

1. Clone o repositório:
   ```bash
   git clone https://github.com/gabrielmco/Agenica-de-Modelos.git
   ```
2. Abra o `index.html` em seu servidor local (Live Server / Vercel preview).

## ⚡ Deploy (Vercel)

Este projeto foi otimizado para a **Vercel**. 

1. Acesse o dashboard da Vercel.
2. Importe este repositório.
3. Clique em **Deploy**.
